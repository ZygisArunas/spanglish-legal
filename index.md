# Privacy Policy

Last updated: 2026-04-22

This Privacy Policy explains how Spanglish Translator Pro ("Extension") and its backend service ("Service") process data.

## 1. Who We Are

Product: Spanglish Translator Pro  
Contact: info@spanglishtranslator.app

## 2. Data We Process

### A. Text You Submit

- Text typed in the extension popup
- Text you actively select on web pages for inline translation

Purpose:

- Provide translation results

Processing:

- Text is sent to our backend API and then to OpenAI to generate translation output.

Retention:

- Translation requests are not intentionally stored as long-term user history.
- Short-lived in-memory caching may be used for performance.

### B. Technical and Plan Data

- Generated API key identifier (stored locally in extension storage)
- Install fingerprint (stored locally; may be stored server-side for client reuse/rate protection)
- API key hash (server-side)
- Usage counters (daily/monthly request counts)
- Plan state (free/pro)

Purpose:

- Authentication, abuse prevention, quota enforcement, and plan access control

### C. Billing Data

- Billing is processed via Stripe Checkout
- We do not collect or store raw card numbers in the extension
- Stripe may send subscription event metadata needed to activate/deactivate Pro access

## 3. How We Use Data

We use data to:

- Deliver translations
- Enforce plan limits
- Prevent abuse
- Support billing and subscription status

We do not sell personal data and do not use collected data for advertising.

## 4. Third-Party Processors

- OpenAI (translation processing)
- Stripe (payment and subscription processing)
- Hosting/database provider used to run backend infrastructure

## 5. Data Sharing

We share data only with providers required to run the product, and only as needed to provide the service.

## 6. Security

We apply reasonable technical measures including API-key authentication, CORS restrictions, and server-side validation/rate controls.

## 7. Your Choices

You can:

- Uninstall the extension at any time
- Clear extension local data from Chrome extension settings
- Contact support to request deletion of associated service data where applicable

## 8. Children's Privacy

The Service is not directed to children under 13.

## 9. Changes to This Policy

We may update this policy. The latest version date appears at the top.

## 10. Contact

For privacy requests or questions, contact: info@spanglishtranslator.app
